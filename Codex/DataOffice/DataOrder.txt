
============================================================================================================
ІНСТРУКЦІЯ ДО ОБ'ЄКТУ: 'DataOrder.py' / 'DataOrder.exe'
======================================================

Призначення:
- пакетна рекурсивна обробка папки з Office-файлами, зображеннями та PDF.

Що робить (тумблери):
1. Очищення temp/lock:
   - переносить у кошик службові Office-файли '~$*', '.~lock.*' та відповідні '.tmp'.
2. Новіший формат:
   - конвертує DOC -> DOCX;
   - конвертує XLS -> XLSX;
   - старі DOC/XLS після успішної обробки переносить у кошик.
3. Зображення -> PDF:
   - конвертує JPG/JPEG/PNG/TIF/TIFF/BMP у PDF.
4. PDF -> PDF with OCR:
   - проганяє PDF через OCR і робить PDF з розпізнаним текстом.
5. Обробка PDF:
   - стискає PDF;
   - прибирає '/Title' з PDF-метаданих.
6. PDF -> Word:
   - конвертує PDF у DOCX через Microsoft Word.

Як користуватися:
1. Запустити.
2. Задати координати. Пуск.
3. Дочекатися завершення прогресу.

Нюанси:
- етапи можна запускати або усі разом, або в комбінації або поодинці.
- усі файли без винятку файли очищені кодом — потрапляють до «Кошика».
- для OCR-етапу потрібні: Tesseract ("https://github.com/UB-Mannheim/tesseract/wiki") та Ghostscript ("https://ghostscript.com/releases/gsdnld.html").
- для Office-конвертацій та PDF -> DOCX необхідно мати встановлений Microsoft Office.
