
============================================================================================================
ІНСТРУКЦІЯ ДО ОБ'ЄКТУ: 'AuditHSW.py' / 'AuditHSW.exe'
======================================================

Призначення:
- аудит поточного Windows-ПК: апаратна частина, ОС, мережа, програми, локальні користувачі, ESET, ліцензійні сліди.

Що робить:
- збирає серійний номер, BIOS, build Windows, мережеві адаптери, диски, пам'ять, CPU, GPU, NPU;
- читає локальних користувачів і групи; список встановлених програм;
- витягує доступні ліцензійні сигнатури Windows;
- формує текстовий лог-звіт.

Як користуватися:
1. Запустити. Пуск.
2. Дочекатися завершення прогресу.

Нюанси:
- це Windows-only скрипт.
- назва лог-звіту формується, як: 'Log[A/U] - <SerialNumber>.log'. // 'A' — позначає, запуск з правами адміністратора, 'U' — користувацькими. // Права запуску: у більшості випадків не мають значення — код коректно працює навіть зі звичайними користувацькими правами. Єдиний виняток — сектор програм чужих облікових записів, де адмін-права забезпечують дещо повнішу звітність.
